Quels canaux PSG sont disponibles dans votre fichier EDF ?
Un enregistrement PSG complet contient 5 canaux : deux EEG, deux EOG et un EMG. Les appareils ambulatoires portables ne capturent souvent que 2 canaux EEG. Plus de canaux = meilleure précision, surtout pour détecter le sommeil REM.
Combien de stades de sommeil souhaitez-vous classifier ?
La classification en 3 classes (Wake, NREM, REM) est très robuste. L'ajout des stades N1/N2/N3 (5 classes) augmente l'ambiguïté (l'accord inter-juges humain pour N1 n'est que de ~40%).
Chargez le fichier PSG de votre patient
Format accepté : EDF (European Data Format). Le fichier est analysé localement via le serveur Flask. Les données ne sont jamais envoyées vers un serveur externe. Durée typique d'enregistrement : 7–9 heures = environ 1000–1100 époques de 30 secondes.
